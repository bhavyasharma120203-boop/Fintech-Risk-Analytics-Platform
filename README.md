# Fintech Risk Analytics Platform

## Project Overview

This project develops an end-to-end fintech analytics solution to identify loan default risk, segment customers based on risk profiles, forecast future portfolio performance, and support data-driven lending decisions.

The project simulates a real-world fintech use case where financial institutions need to assess borrower risk, minimize defaults, and improve lending strategies through analytics and machine learning.

---

## Business Problem

Loan defaults create significant financial losses for banks and fintech companies. Traditional lending decisions often fail to identify high-risk borrowers early.

This project aims to answer:

* Which customers are most likely to default?
* What factors influence loan repayment behavior?
* How can customers be segmented by risk level?
* What future default trends should lenders expect?

---

## Dataset

The analysis uses customer, loan, repayment, and transaction-level data containing information such as:

* Customer demographics
* Employment and income details
* Loan characteristics
* Repayment history
* Transaction behavior
